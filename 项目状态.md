## 技术决策记录

### 决策1：向量数据库选型
**日期**：2026-06-06
**决策**：使用ChromaDB
**理由**：
- 轻量级，无需独立部署
- 支持本地持久化
- Python原生支持
- 社区活跃

### 决策2：嵌入模型选型
**日期**：2026-06-06
**决策**：使用sentence-transformers (all-MiniLM-L6-v2)
**理由**：
- 支持本地运行
- 模型体积小（~80MB）
- 384维向量，性能与体积平衡
- 也支持API模式（通过OpenAI兼容协议）

### 决策3：LLM接口设计
**日期**：2026-06-06
**决策**：使用OpenAI兼容协议（直接用openai库）
**理由**：
- DeepSeek、OpenAI等主流服务都兼容
- 无需额外依赖（litellm等）
- 支持流式输出

### 决策4：HuggingFace镜像
**日期**：2026-06-06
**决策**：默认使用官方源，镜像配置注释掉
**理由**：
- hf-mirror.com 镜像不稳定
- 有VPN时官方源更可靠
- 用户可自行取消注释启用镜像

## 使用方法

### 命令行

```bash
# 查看配置状态
python main.py config

# 索引文档
python main.py index ./data/documents

# 单次提问
python main.py ask "你的问题"

# 交互式对话
python main.py chat

# 启动Web界面
python main.py web
```

### 配置

编辑 `.env` 文件配置 LLM API：

```env
LLM_BASE_URL=https://api.deepseek.com
LLM_API_KEY=your-api-key
LLM_MODEL=deepseek-v4-flash
```
