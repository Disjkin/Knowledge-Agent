# 私有文件检索Agent - 架构设计文档

## 1. 系统架构概述

### 1.1 整体架构

```
┌─────────────────────────────────────────────────────────────┐
│                    用户界面层 (Streamlit)                      │
├─────────────────────────────────────────────────────────────┤
│                    Agent 核心层                               │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐    │
│  │ 查询理解  │  │ 检索编排  │  │ 上下文组装 │  │ 回答生成  │    │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘    │
├─────────────────────────────────────────────────────────────┤
│                    服务层                                     │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐    │
│  │ 文档加载  │  │ 文本分割  │  │ 向量检索  │  │ LLM 接口 │    │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘    │
├─────────────────────────────────────────────────────────────┤
│                    数据层                                     │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐                   │
│  │ 文件系统  │  │ ChromaDB │  │ 配置文件  │                   │
│  └──────────┘  └──────────┘  └──────────┘                   │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 数据流

```
文档导入流程：
文件 → 文档加载器 → 文本分割 → 嵌入向量 → ChromaDB存储

查询流程：
用户问题 → 查询理解 → 向量检索 → 上下文组装 → LLM推理 → 回答
```

### 2.7 Agent核心 (Agent)

**职责**：编排整个RAG流程

```python
class Agent:
    """RAG Agent核心"""

    def __init__(self, retriever: Retriever, llm: LLMInterface):
        self.retriever = retriever
        self.llm = llm
        self.memory = ConversationBufferMemory()

    def ask(self, question: str) -> str:
        """回答问题"""
        # 1. 检索相关文档
        docs = self.retriever.retrieve(question)

        # 2. 组装上下文
        context = self._build_context(docs)

        # 3. 构建提示词
        prompt = self._build_prompt(question, context)

        # 4. 调用LLM
        response = self.llm.chat(prompt)

        # 5. 保存对话历史
        self.memory.add(question, response)

        return response
```

## 3. 数据模型

### 3.1 Document

```python
@dataclass
class Document:
    content: str           # 文本内容
    metadata: dict         # 元数据
    # metadata包含：
    # - source: 文件路径
    # - file_type: 文件类型
    # - page: 页码（PDF）
    # - chunk_index: 分块索引
```

### 3.2 配置模型

```yaml
# config.yaml
embedding:
  model_name: "all-MiniLM-L6-v2"
  device: "cpu"

vector_store:
  persist_directory: "./data/chroma"
  collection_name: "knowledge"

llm:
  provider: "openai"
  model: "gpt-4"
  api_key: "${OPENAI_API_KEY}"
  temperature: 0.7

text_splitter:
  chunk_size: 500
  chunk_overlap: 50

retrieval:
  top_k: 5
```

## 6. 安全设计

### 6.1 数据安全
- 所有文档本地存储，不上传外部服务
- 向量数据库本地持久化
- 对话历史本地存储

### 6.2 API安全
- API密钥存储在本地配置文件
- 支持环境变量配置
- 不记录敏感信息到日志
